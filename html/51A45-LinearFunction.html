linear function


Let 𝒮1=(𝒫1,ℒ1) and 𝒮2=(𝒫2,ℒ2) be two near-linear spaces.

Definition. A linear function from 𝒮1 to 𝒮2 is a mapping on the points that sends lines of 𝒮1 to lines of 𝒮2. In other words, a linear function is a function σ:𝒫1→𝒫2 such that

σ⁢(ℓ)∈ℒ2⁢ for every ⁢ℓ∈ℒ1.

Here, σ⁢(ℓ) is the set {σ⁢(P)∣P∈ℓ}. A linear function is also called a homomorphism.

When both 𝒮1 and 𝒮2 are linear spaces, then σ being a linear function is equvalent to saying that P,Q are collinear iff σ⁢(P),σ⁢(Q) are collinear.

If 𝒮1 is a linear space, then so is (σ⁢(𝒫1),σ⁢(ℒ1)). This shows that if σ:𝒮1→𝒮1 is onto, 𝒮2 is a linear space if 𝒮1 is.

Let σ:𝒮1→𝒮2 be a one-to-one linear function. If points P1≠P2 lie on line ℓ, then σ⁢(P1)≠σ⁢(P2) lie on σ⁢(ℓ). This also shows that three collinear points in 𝒮1 are mapped to three collinear points in 𝒮2. In addition, we have

|ℓ|=|σ⁢(ℓ)| for any line ℓ in 𝒮1.

Definition. When σ:𝒮1→𝒮2 is a bijection whose inverse σ-1 is also linear, we say that σ is an isomorphism. When 𝒮1=𝒮2=𝒮, we call σ an automorphism, or more commonly among geometers, a collineation, of the space 𝒮.

Suppose σ:𝒮1→𝒮2 is an isomorphism. For every point P, let P* be the set of all lines passing through P. Then

|P*|=|σ⁢(P)*| for any point P in 𝒮1.

It is possible to have a bijective linear function whose inverse is not linear. For example, let 𝒮1 be the space with two points P,Q with no lines, and 𝒮2 the space with the same two points with line {P,Q}. Then the identity function on {P,Q} is a bijective linear function whose inverse is not linear. On the other hand, if the both spaces are linear, then the inverse is always linear.

Remark. The usage of the term “linear function” differs from its more usual meaning as a linear transformation between vector spaces in the study of linear algebra.
