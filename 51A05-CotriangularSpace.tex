A \ti{cotriangular space} is a \PMlinkid{partial linear space}{PartialLinearSpace} $(P,L)$ in which 
\begin{itemize}
\item each line contains exactly three points, and
\item given a point not
on a line, the point is collinear with zero or two points of the line.
\end{itemize}
